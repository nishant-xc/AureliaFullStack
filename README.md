# AureliaFullStack

A production-ready food delivery backend built with **Node.js**, **Express.js**, and **PostgreSQL**.

## Features

- User Authentication (JWT)
- Restaurant Management
- Menu & Categories
- Cart Management
- Order Management
- Payment Module
- Delivery Partner Module
- Reviews & Ratings
- Notifications
- Address Management
- Admin Module
- RESTful API
- Input Validation
- Rate Limiting
- Automated Tests
